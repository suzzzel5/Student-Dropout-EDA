# Student Dropout EDA

Project Overview

The dataset is about online student learning behavior in the Arab world — specifically built to understand why students drop out of online courses.

# The Core Problem 
The platform noticed that many students were silently disengaging — logging in less, submitting fewer assignments, and eventually disappearing. So they built a dropout risk score for each student and categorized them into three groups

- Active — engaged, completing work, logging in regularly
- At-risk — showing early warning signs of disengagement
- Dropped — effectively quit the platform

Student dropout is a critical challenge in online education. This project dives into a dataset of 5,000 students to uncover patterns, correlations, and risk indicators associated with dropout behavior — using only Python and standard data science libraries.
This is a learning project focused on mastering EDA fundamentals step by step.
